# Introduction au traitement d'image
Pour le projet dans le cours "Introduction au traitement d'image", je propose la détection et anonymisation des visages de célébrités avec l'algorithme Haar Cascades

# Algorithme
Haar Cascades

# Données
Les données sont des photos de célébrités. Elles incluent des photos où les célébrités sont seuls, en groupe, avec des lunettes de soleil et sans.
