# Introduction au traitement d'image

# Description
Pour le projet dans le cours "Introduction au traitement d'image", je propose la détection et anonymisation des visages de célébrités avec l'algorithme Haar Cascades.

# Prérequis
- La librairie [OpenCV (cv2)](https://www.geeksforgeeks.org/installation-guide/how-to-install-opencv-for-visual-studio-code-and-python/ )
- pathlib
- matplotlib

# Algorithme
Haar Cascades, proposé par Viola et Jones, est une méthode basée sur le machine learning et peut détecter des visages dans les images.

# Données
Les données sont des photos de célébrités. Elles incluent des photos où les célébrités sont seules, en groupe, regardant la caméra et regardant sur le côté, et avec des lunettes de soleil et sans.
